# 🇦🇪 Exploring UAE - Interactive Quiz App

An interactive web application designed to test and enhance your knowledge about the United Arab Emirates through engaging quizzes covering geography, history, culture, and general knowledge.

## ✨ Features

- 🎯 **50+ Questions** across 4 categories
- ⏱️ **Timed Questions** - 30 seconds per question
- 📊 **Real-time Scoring** - Track your progress instantly
- 🏆 **Leaderboard** - Compete with other players
- 🎨 **Beautiful UI** - Modern, responsive design
- 📱 **Mobile Friendly** - Works on all devices
- 🌐 **Bilingual Ready** - Supports English and Arabic
- 🎉 **Gamification** - Badges, levels, and achievements
- 📚 **Educational** - Learn interesting facts about UAE

## 📋 Categories

1. **Geography** - Learn about UAE's landscapes, cities, and landmarks
2. **History** - Discover the UAE's rich historical heritage
3. **Culture & Traditions** - Explore Emirati culture and customs
4. **General Knowledge** - Test your overall UAE knowledge

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/ishanviverma24/exploring-uae.git

# Navigate to the project directory
cd exploring-uae

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:3000`

6. Share your results with friends
7. Check the leaderboard to see how you rank
